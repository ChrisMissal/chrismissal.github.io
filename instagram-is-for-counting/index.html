I have an Instagram account, but I only use it for counting.

There are only a few "rules" that I'm doing my best to follow:

  1. Photo contains integers, no skips, no duplicates
  2. Don't take photos of the same thing twice. No duplicates on things like playing cards or pool balls.
  3. Any language counts! (Arabic numerals, roman numerals, English text, Spanish text, etc)
